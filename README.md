# Drug Discovery ML (Compound–Protein Binding Prediction)

This project applies **bioinformatics and machine learning** to predict interactions between chemical compounds and target proteins.  
By leveraging public drug–protein binding datasets, it accelerates the **early stages of drug discovery**, reducing reliance on costly and time-consuming lab experiments.

---

## 🚀 Features
- Predicts compound–protein binding affinity using deep learning models (GraphDTA, CNN, etc.).  
- Utilizes graph-based molecular representations and protein sequence embeddings.  
- Trains on benchmark datasets like **DAVIS** and **KIBA**.  
- Outputs performance metrics (MSE, CI, Pearson’s r) to evaluate predictive power.
